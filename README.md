# CBSE School Data API

### API Endpoints

GET /api/schools/{affiliatedId}
returns info about that school info

GET /api/states
returns states of India (According to SARAS data)

GET /api/states/{state}
returns districts of that state

GET /api/states/{state}/{district}
returns the schools in that district.
